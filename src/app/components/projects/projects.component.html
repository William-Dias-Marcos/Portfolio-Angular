Weather Forecast

Visualizar GitHub

Weather Forecast representa uma aplicação Angular que fornece previsões meteorológicas em tempo real para cidades globais. Usando TypeScript, HTML, CSS e a API do OpenWeather, oferece informações confiáveis. Destaca-se pela confiabilidade, graças aos testes com o Jasmine, proporcionando uma experiência confiável para atualizações futuras.

  • Angular
  • TypeScript
  • CSS
  • HTML
  • API
  • Jasmine

Word Currencies

Visualizar GitHub

O Word Currencies é um conversor de moedas desenvolvido com TypeScript, HTML, Bootstrap e Angular. A aplicação utiliza serviços, pipes, diretivas e módulos do Angular para melhorar a organização e a funcionalidade. Ele consome a API da AwesomeAPI para obter os dados das moedas e permite que os usuários convertam moedas facilmente.

  • Angular
  • TypeScript
  • Bootstrap
  • HTML
  • API
